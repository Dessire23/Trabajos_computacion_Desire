Sección 1: Introducción, Variables y Tipos de Datos

Python es un lenguaje de programación de alto nivel, conocido por su sintaxis clara y legible. Es ideal para principiantes y profesionales en ciencia de datos, desarrollo web y automatización.

1.1 Tu primer programa

Para mostrar texto en la pantalla, usamos la función print():

# Esto es un comentario. El código de abajo imprime un texto
print("¡Hola, Mundo!")

1.2 Variables y Tipos Base

Las variables guardan información. No necesitas declarar el tipo de datos de forma explícita, Python lo detecta automáticamente:

nombre = "Carlos"      # Tipo de dato Texto (String / str)
edad = 25               # Tipo Entero (Integer / int)
precio = 19.99          # Tipo Decimal (Float / float)
es_activo = True        # Tipo Booleano (Boolean / bool)

# Puedes combinarlas en un print usando "f-strings"
print(f"Hola, {nombre}. Tienes {edad} años.")

Sección 2: Control de Flujo (Condicionales y Bucles)

Para tomar decisiones o repetir tareas en la programación, usamos estructuras de control de flujo.

2.1 Condicionales (if, elif, else)

Python utiliza la indentación (espacios al inicio de la línea) para definir bloques de código.

hora = 14

if hora < 12:
    print("Buenos días")
elif hora < 20:
    print("Buenas tardes")
else:
    print("Buenas noches")

2.2 El Bucle for

Se utiliza para iterar sobre una secuencia (como una lista o un rango de números).

# Imprimir los números del 1 al 3
for i in range(1, 4):
    print(f"Número: {i}")

2.3 El Bucle while

Se ejecuta mientras una condición se mantenga verdadera (True).

contador = 1
while contador <= 3:
    print(f"Contando: {contador}")
    contador = contador + 1

Sección 3: Colecciones de Datos y Funciones

Aprende a agrupar información compleja y a encapsular código para poder reutilizarlo.

3.1 Listas

Son colecciones ordenadas y mutables de elementos (pueden ser modificadas).

frutas = ["manzana", "banana", "cereza"]

# Acceder por índice (comienzan en 0)
print(frutas[0])  # Salida: manzana

# Agregar un elemento
frutas.append("naranja")

3.2 Funciones

Una función es un bloque de código reutilizable. Se define con la palabra clave def.

def saludar(usuario):
    mensaje = f"¡Bienvenido de nuevo, {usuario}!"
    return mensaje

# Llamar o ejecutar la función
mi_saludo = saludar("Lucía")
print(mi_saludo)
